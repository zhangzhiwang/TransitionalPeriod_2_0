# 网站设计图 · 网站新闻中心

> 风格基准：苹果 Newsroom / 博客列表 — 清晰层级、大标题、日期次要、图文节奏克制。  
> 导航：新闻中心为当前选中项。

---

## 1. 页面信息架构

```mermaid
flowchart TB
  Nav[顶栏导航]
  subgraph List["新闻列表页"]
    Hero[页面标题区]
    Featured[头条精选 1 则]
    Feed[新闻列表/网格]
    Filter[分类筛选 Tab]
  end
  subgraph Article["新闻详情页"]
    Title[标题+日期+摘要]
    Cover[全宽封面]
    Body[正文排版]
    Related[相关阅读]
  end
  Nav --> List
  Feed --> Article
  Featured --> Article
```

## 4. 视觉规范

| 维度 | 规范 |
|------|------|
| 列表背景 | `#F5F5F7`；条目区白底或统一浅底 |
| 日期 | `#86868B`，小字号 |
| 标题悬停 | 变为链接蓝或下划线，无夸张动效 |
| 正文 | 行宽舒适；图片全宽于内容栏内，圆角可选 0–8px |

---

## 5. 移动端

- 头条与列表均单列；Tab 可横向滚动。
- 详情页边距 16–20px；封面全宽。

---

## 6. 交互要点

1. Tab 切换使用 `startTransition` 友好的列表更新（实现阶段）。  
2. 分页优先「显示更多」无限滚动，兼顾客厅式浏览。  
3. SEO：详情页独立 URL、语义化标题层级。
