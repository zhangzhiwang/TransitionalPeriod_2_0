## 4. 关键字段说明

### 4.1 mall_user

- `password_hash`：落库密文，禁止明文（架构要求）。
- `nickname` / `avatar_url`：可来自微信授权回调更新。
- `wechat_openid`：可选绑定，唯一可空。

### 4.2 mall_product

- `recommend=1`：参与首页推荐列表。
- `stock`：加购与下单校验库存。

### 4.3 mall_cart_item

- `(user_id, product_id)` 唯一：同一商品数量累加。
- `selected`：结算勾选 / 全选。

### 4.4 mall_user_address

- `default_address`：同一用户业务上最多一条为 1（设默认时清其他）。

### 4.5 mall_order / mall_order_item

- 订单状态：`PENDING_PAY` / `PAID` / `CANCELLED` / `CLOSED`。
- 收货人、电话、地址及商品名/图/单价在下单时 **快照**，保证订单列表历史正确。
- 微信支付字段：`wechat_prepay_id`、`wechat_transaction_id`、`pay_time`。
