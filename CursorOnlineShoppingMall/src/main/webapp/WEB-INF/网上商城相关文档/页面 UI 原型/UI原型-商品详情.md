## 1. 页面信息

| 项 | 说明 |
|----|------|
| 路由建议 | `/product/{id}` |
| 访问条件 | 需登录 |
| 加入购物车 | 写入购物车列表 |
| 立即购买 | 直接跳转订单确认页 |

---

## 2. 信息架构

```mermaid
flowchart TB
  Nav[返回]
  Img[商品大图]
  Info[名称 / 价格 / 库存]
  Bar[加入购物车 | 立即购买]
  Nav --> Img --> Info --> Bar
```

---

## 4. 交互说明

| 操作 | 行为 |
|------|------|
| 加入购物车 | 添加成功 toast；可停留本页 |
| 立即购买 | 携带当前商品跳转订单确认页 |
| 返回 | 回上一页（首页/分类商品列表等） |

---

## 5. 组件要点

- 底栏双按钮等高，立即购买视觉权重更高
- 库存为 0 时两按钮禁用并提示缺货
