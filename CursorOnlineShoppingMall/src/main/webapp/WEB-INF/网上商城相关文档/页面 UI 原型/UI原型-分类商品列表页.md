---

## 1. 页面信息

| 项 | 说明 |
|----|------|
| 路由建议 | `/category/{id}/products` |
| 访问条件 | 需登录 |
| 展示字段 | 名称、图片、价格、库存、加入购物车 |
| 点击商品 | 进入商品详情 |

## 5. 交互说明

| 操作 | 行为 |
|------|------|
| 点击商品图/名 | 进入商品详情 |
| 加入购物车 | 加购成功提示 |
| 返回 | 回分类页或分类列表页 |
