## 3. 底部导航（登录后主站）

未登录访问首页/分类/购物车/我的时，统一跳转登录页。

```
┌──────────┬──────────┬──────────┬──────────┐
│   首页   │   分类   │  购物车  │   我的   │
└──────────┴──────────┴──────────┴──────────┘
```

- 选中态：图标与文字为 `#E4393C`
- 未选中：`#666666`
- 固定底部，内容区预留底部安全高度
