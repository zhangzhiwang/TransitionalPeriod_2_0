|----|------|

## 4. 交互说明

| 操作 | 行为 |
|------|------|
| 点击登录 | 校验非空 → 请求登录 → 成功进首页；失败展示错误信息 |
| 点击去注册 | 跳转注册页 |
| 回车 | 同点击登录 |

---

## 5. 组件要点

- 输入框：白底、浅灰边框、圆角 2–4px
- 主按钮：全宽、高 40–44px、`#E4393C`、白字
- 错误文案：`#F56C6C`，登录失败时可见
